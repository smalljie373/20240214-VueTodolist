# vite-todolist

## 使用技術
- Vue3.0
- Tailwind CSS 3.0
- node 18.14.0
- axios 1.4.0
- sweetalert2 5.0.5


## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
